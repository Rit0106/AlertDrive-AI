
# AlertDrive-AI 🚗💤

AlertDrive-AI is an AI-powered Driver Drowsiness Detection and Safety Monitoring System built using YOLOv8, OpenCV, and Python. The system performs real-time eye-state and yawn detection through computer vision techniques to identify driver fatigue and trigger intelligent alert mechanisms for accident prevention.

This project aims to enhance road safety by continuously monitoring driver behavior, generating emergency alerts during prolonged drowsiness, and serving as a foundation for future MERN stack integration, analytics dashboards, and real-time monitoring solutions.

### Tech Stack

* Python
* YOLOv8
* OpenCV
* Computer Vision
* Machine Learning
* MERN Stack (Planned)

### Features

* Real-time eye closure detection
* Yawn detection system
* Driver fatigue monitoring
* Emergency alert triggering
* Live webcam processing
* AI-based safety assistance system
